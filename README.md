Il file PreparazioneDatasets è un notebook, eseguito in ambiete Google Colab per ottenere tutti i dataset degli esperimenti.
Tutti gli script sono stati riposti in cartelle dedicate divise in funzione del Job 3.1 e 3.2, e in funzione del dataset su cui sono stati utilizzati. 
Per entrambi i Job sono riportati i comandi per riprodurre tutti i test riportati all'interno della relazione in dei file README dedicati.
Per facilitare l'esecuzione dei test da parte dei valutatori, sono stati forniti script separati con i path hardcoded per i vari ambienti (locale vs AWS), sebbene in un ambiente di produzione si utilizzerebbe un unico script parametrizzato.
Nella cartella ScriptEsecuzione ci sta uno script bash che permette l'esecuzione di tutti gli script passando da riga di comando l'input e l'outupt. Le istruzioni sono presenti un un README dedicato.
